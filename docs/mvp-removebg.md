# 移除图片背景工具 - MVP 需求文档

## 1. 项目概述

- **项目名称**：RemoveBG Mini
- **项目类型**：在线图片处理工具（纯前端）
- **核心功能**：用户上传图片，自动移除背景，返回透明背景图片
- **目标用户**：需要快速去除图片背景的非专业用户（设计师、自媒体从业者等）
- **部署平台**：Cloudflare Pages

## 2. 技术方案

### 技术栈

- **前端框架**：React 18 + Vite
- **样式**：Tailwind CSS
- **部署**：Cloudflare Pages
- **图片处理API**：remove.bg（有免费额度：50张/月）

### 架构设计

- 纯前端架构，无需后端服务器
- 图片全程在用户浏览器内存中处理，不经过服务器存储
- Cloudflare Pages 托管静态资源
- API Key 通过环境变量注入

### 数据流程

```
用户选择图片 → FileReader 读取到内存 → 转 Base64 → 调用 remove.bg API → 返回透明图 → 浏览器预览/下载
```

## 3. 功能需求

### MVP 必须功能 (P0)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽上传和点击选择，支持 jpg/png/webp | P0 |
| 图片预览 | 上传后显示原图预览 | P0 |
| 背景移除 | 调用 remove.bg API 处理图片 | P0 |
| 结果预览 | 显示处理后的透明图预览 | P0 |
| 图片下载 | 一键下载处理结果（PNG格式） | P0 |
| 加载状态 | 处理中显示 loading 状态 | P0 |
| 错误处理 | API 失败时显示友好错误提示 | P0 |

### 后续迭代功能 (P1)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 对比模式 | 原图/结果左右对比 | P1 |
| 背景替换 | 可选择纯色背景 | P1 |
| 批量处理 | 一次处理多张图片 | P1 |

## 4. 非功能性需求

### 性能要求

- 首屏加载 < 3秒（不含模型/资源）
- 图片处理时间取决于 remove.bg API 响应速度
- 支持最大图片尺寸：4MB

### 兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge 最近2个版本）
- 移动端响应式适配

### 安全

- API Key 存储在 Cloudflare 环境变量，不暴露在前端代码
- 图片数据不经过任何服务器

## 5. UI/UX 设计方向

### 页面结构（单页面）

- 顶部：产品名称 + 简短说明
- 中间：上传区域（主交互区）
- 底部：版权信息

### 交互流程

1. 用户进入页面 → 显示上传区域
2. 上传图片 → 自动触发处理
3. 处理中 → 显示 loading
4. 处理完成 → 显示原图/结果对比 + 下载按钮
5. 可继续上传新图片

### 设计风格

- 简洁现代，留白充足
- 主色调：紫色系（remove.bg 品牌色相近）
- 支持明暗主题（跟随系统）

## 6. 项目里程碑

| 阶段 | 内容 | 验收标准 |
|------|------|----------|
| MVP | 完成基础上传、处理、下载功能 | 能完整处理一张图片并下载 |
| 体验优化 | 加载状态、错误处理、UI优化 | 用户可流畅完成整个流程 |
| 正式上线 | 部署到 Cloudflare，绑定域名 | 外部可访问使用 |
